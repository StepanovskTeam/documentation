# documentation
Markdown (.md) файлы, содержащие технические документации для всех игр Stepanovsk. Изначально созданы для того чтобы разработчик не запутался в своих же не очень логичных решениях, а также чтобы игроки в случае если захотят полезть внутрь - поняли что этот конченный имел ввиду под тем или иным скорбордом, или зачем та или иная функция вообще нужна.
